# ТЗ для слейва (короткая версия)

## Modbus
- RS485, Modbus RTU, 9600 8N1
- Мастер читает 0x03, пишет уставки 0x10

## Телеметрия (start_register + offset)
- +0 AIR_TEMP (x10)
- +1 AIR_HUM (x10)
- +2 WATER_RAIL (x10)
- +3 WATER_GROW (x10)
- +4 WATER_UNDERTRAY (x10)
- +5 WATER_UPPER_HEAT (x10)
- +6 WINDOWS_POS_A (x10)
- +7 WINDOWS_POS_B (x10)
- +8 CURTAIN_POS (x10)

## Частоты на слейве
- Температуры/влажность: 5 сек
- Позиционеры (A/B/шторы): 200 мс

## Автономный режим
- Heartbeat = любой валидный Modbus-запрос мастера
- Нет запросов 30 сек -> AUTONOMOUS
- На входе в AUTONOMOUS: форточки закрыть полностью
- Локальные автономные уставки/расписание зашиты в прошивку и мастером не меняются
- Возврат в REMOTE: 3 подряд успешных цикла связи
- После возврата: управление только по уставкам мастера

## Свет (REMOTE)
- До 4 расписаний
- Несколько расписаний могут быть активны одновременно
- Логика включения: OR по активным интервалам
- Интервалы через полночь поддерживаются (пример 20:00-01:00)
- Расписания одинаковы для всех дней недели

## Свет (AUTONOMOUS)
- Отдельное фиксированное автономное расписание

## Мини-приемка
1. Чтение +0..+8 без ошибок
2. Позиционеры обновляются по 200 мс локально
3. Переходы REMOTE<->AUTONOMOUS по таймингам корректны
4. 4 расписания REMOTE и интервал через полночь работают корректно
